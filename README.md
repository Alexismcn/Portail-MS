# Portail Marseille Simulation — Prototype

Structure minimale du prototype du portail en HTML/CSS/JS.

- Ouvrir `index.html` dans un navigateur pour voir l'accueil.
- `tools.html` contient l'annuaire des outils avec recherche/filtre.
- `explorer.html` contient un explorateur hiérarchique des ressources.

Ajouter ou modifier des outils dans `data/tools.json`. Les logos référencés sont dans `Images/logomsBlanc.png` et `Images/logomsbleu.png`.

Pour développer : utiliser un serveur simple (ex: `python -m http.server 8000`) pour éviter les problèmes de fetch local.
